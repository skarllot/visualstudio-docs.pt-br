# <a name="extending-projects"></a>Estendendo projetos
Projetos e soluções são as maneiras que Visual Studio organiza o código e arquivos de recursos em unidades de compilação e implantação. Você pode encontrar mais informações sobre projetos na [projetos (SDK do Visual Studio)](../extensibility/extending-projects.md).  
  
 Você pode criar seus próprios tipos de projeto com o SDK do Visual Studio e a estrutura de pacote gerenciado para projetos que você pode baixar em [estrutura de pacote gerenciado para projetos](http://mpfproj12.codeplex.com/). Para entender como os projetos personalizados são implementados, consulte [nova geração de projeto: sob o escopo, parte&1;](../extensibility/internals/new-project-generation-under-the-hood-part-one.md) e [nova geração de projeto: sob o escopo, parte dois](../extensibility/internals/new-project-generation-under-the-hood-part-two.md).  
  
 Os tópicos nesta seção descrevem como criar projetos personalizados e como gerenciar diferentes tipos de solução do Visual Studio.
