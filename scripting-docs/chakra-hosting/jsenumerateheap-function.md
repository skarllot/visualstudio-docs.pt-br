# <a name="jsenumerateheap-function"></a>Função JsEnumerateHeap
Enumera o heap do contexto atual.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
STDAPI_(JsErrorCode) JsEnumerateHeap(  
   _Out_ IActiveScriptProfilerHeapEnum **enumerator  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `enumerator`  
 O enumerador de heap.  
  
## <a name="return-value"></a>Valor de retorno  
 O código `JsNoError` se a operação foi bem-sucedida, caso contrário, um código de falha.  
  
## <a name="remarks"></a>Comentários  
 Enquanto o heap está sendo enumerado, o contexto atual não pode ser removido e todas as chamadas para modificar o estado do contexto falham até o enumerador de heap ser liberado.  
  
 Exige um contexto de script ativo.  
  
 Essa API tem suporte em aplicativos de área de trabalho, mas não tem suporte em Aplicativos da Windows Store.  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** jsrt.h
